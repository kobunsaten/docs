# 眼睛与色彩

# 色彩的来源——光

# 光的本质

  1. 光是色彩的重要来源，没有光就没有色彩。

  2. 分为人造光源和自然光源。

  3. 光的物理性质决定于振幅和波长：波长决定色相，振幅决定明暗。波长<400nm为紫外线，波长>700nm为红外线。

  4. 声音的本质也是波，和光可以在艺术形式上的辩证统一和结合。

# 光的属性

色彩只是对光敏感的结果，也就是说光具有显色性。

光的显色性：同一个颜色样品在不同光源下可能使人眼产生不同的色彩感觉，日光下的颜色是最准确的，因此可以用日光做标准，评判人工光源的显色性。

肤色：柔和白光照射下为正常肤色；夕阳照射下为橙黄色。（体现为色温）

不同光源色可以形成不同的色调气氛。主要影响因素为亮度、距离、传播媒介。

光源色对物体色的影响主要表现在物体的亮部。影响强度排行为：红白绿蓝青紫。

物体的固有色一般是不变的，但会受到邻近物体的颜色（环境色）的影响。环境色影响主要在暗部。一般来说，产生环境色物体跃进，被视物体表面越光滑，反射光线越强，则环境色影响越大。

# 光谱

把可见光400nm~700nm分为三段：400nm~500nm为蓝光范围，500nm~600nm为绿光范围，600nm~700nm为红光范围。分别把435.8nm、546.1nm和700nm的蓝、绿、红光成为三原色光。

光谱中的颜色是最纯净的，由于光的吸收和反射，我们在现实世界无法还原出最纯净的色彩，只能无限接近。

拍摄之前往往会拍摄一段含有色板的画面，以便后期调色。除了色板还有一种校色卡叫灰板，用来控制、检验摄影曝光程度、影调、色彩模拟效果。

# 色温

拍摄时需要注意光源环境和摄影机的色温平衡。拍摄前要进行白平衡的校对，有时摄影师通过对白平衡的非正常设置，得到意想不到的奇幻的影像色彩。

# 色彩三属性

# 色相

即色的相性，对应光谱上的一个位置，有着相应的波长。通常来说，色相相差小能够个人平稳而又非同寻常、充满幻想的印象，而色相相差增大则会侧重于现实。

日剧、韩剧给我们清新的感觉，色相差小是重要因素之一；

纪实类侧重于与生活中的自然色彩一致，相对色相差会大一点；

浪漫主义题材的影片特别是爱情片往往给人非常甜美纯情的效果，在影像处理上往往会减少色相差，给画面一种清新的感觉。在色彩设计方面，往往男女主多穿着素色的衣服（多为白色）。

# 明度

明度是指色相的明亮程度，它显示色块的明暗差别。白色是最亮的明度，黑色是最暗的明度。比标准明度淡的为浅色调，比标准明度暗的为深色调。

明度在三要素中有较强的独立性，它可以不带任何色相特征通过黑白灰显现。而色相和纯度则必须依赖一定的明暗来显现。

在创作过程中，需要把有彩色关系抽象为明暗色调，把它看做色彩的骨骼，它是色彩结构的关键。

明度高的色彩给人轻快的感觉，明度低则给人凝重的效果。

明度差小，会表现出高雅平稳；明度差大，会显示出一种力度感，一种活力。

色阶与明度的区别：色阶指画面中色彩的强度，表示画面亮度强弱，色彩本身未改变，与色彩无关。色阶和明度的区别在于，明度在色彩中加白或加黑，改变的是色彩本身。

高色阶的色彩用作近景，低色阶的色彩用作远景，这样可以表现透视效果，从而形成画面深度。色阶高会给人近的感觉，色阶低会给人远的感觉，而高色阶可以吸引人的目光。

# 亚当斯区域曝光法：亚当斯根据影像的亮度由黑到白分成了10个区域，具体如下。

区域 感觉
0区 影像一片漆黑，底片上除了片基本身的色调和灰度外，没有任何可用的密度
1区 影像已非全部漆黑，略有影调，但没有影纹，这是有效临界曝光
2区 影像初步显示出影纹，最暗的部分影调深黑，缺乏纹理
3区 黑暗物体，影调正常，阴暗部分显出了足够的影纹
4区 深色的树叶、石块或景物阴影表现正常；在日光中拍摄人像，阴影部分表现正常
5区 呈中灰色（反射率18%），天空影调较浅，皮肤色调较深，石块呈灰色，木头影调正常
6区 在日光、天空光或人造光中，皮肤影调正常，石块、阳光下的雪景阴影，以及用全色片加浅蓝滤镜拍摄的天空，影调都较浅
7区 皮肤影调很浅，一般物体呈灰色，侧光照射的雪景影调正常
8区 明亮部分影调细腻，有适当影纹，雪景影纹相当明显，人物皮肤上有高光
9区 明亮部分没有影纹，接近于纯白色，与1区的略有影调而没有影纹颇为相似，与10区无明显差别
10区 呈纯白色，画面明亮，有反光

# 纯度

纯度即色彩的鲜艳程度。

纯度与饱和度的不同：纯度的高低是看色彩中的彩色成分和消色成分（黑白色）的比例，与色彩的明度有着有密切的关系，明度变化不管是增大或者减小时纯度都降低，只有明度适中时纯度最高；而饱和度取决于含色成分和消色成分（灰色)的比例。

高纯度色彩给人华丽高雅和成熟的感觉；低纯度给人朴实怀旧、稳重安静的感觉。

纯度取决于白平衡，在日韩小清新风格中，常将白平衡设置为非纯正的白，黑平衡也设置为非正常的黑，给一定的灰度带来不一样的感觉。

# 色彩体系

# 五行色彩学

五行色彩学源自于中国古代万物分类的：金木水火土。它代表了空间观的东西南北中，东主青（青龙、青帝），西主白（白虎、白帝），南主赤（朱雀、赤帝），北主黑（玄武、玄帝），中主黄（黄帝）；对应的是时间观中的春夏秋冬，春青色，夏红色，长夏黄色，秋白色，冬黑色；金为白，木为青，水为黑，火为红，土为黄。

# 普朗体系

由赫伯特·E·艾夫斯提出；最适合色彩理论教学的一种颜料体系。

  1. 色彩属性包括色相、明度、纯度。
  1. 以三原色红黄蓝来分割色轮，三原色不能以其他色彩混成。
  2. 以两个原色等比例混合的色彩为间色形成橙绿紫。
  3. 进一步把原色和间色混合得到黄橙、红橙、红紫、蓝紫、蓝绿及黄绿，把他们有序的放置形成色轮。

后来随着印刷技术的发展，艾夫斯设计了CMYK体系。

# 孟塞尔色立体

由美国美术教育家孟塞尔（A·H·Munsell）于1905年创立的颜色立体模型及由色相、明度、纯度组成的标记（HV/C）标志表面色的方法，是从心理学角度出发的分类。这系统研究减色模式，广泛应用于绘画、印刷、服装等。着重颜色的分类及标定、色彩的逻辑心理与视觉特征。划分了10个基础色相，再细分有100个色相。包括5原色红黄绿蓝紫，5间色黄红、绿黄、蓝绿、蓝紫、红紫。影视调色中色轮根据此原理设计，离中心越远，饱和度越高。

# 奥斯特瓦尔德体系

由化学家奥斯特瓦尔德创立。色彩由彩色（C）、白色（W）、黑色（B）三种成分。色品的排列分布在有CWB的等边三角形中，其中三个顶点为CWB的最大值，当等边三角形高度为100时，三角形内代表任一色品的CWB三数值之和为100。用C/W这一比值表示色彩纯度（奥斯特瓦尔德纯度）。

# CIE系统

混色系统是研究光色成色性质、方法和特点的理论体系，主要指CIE（Commission Internationalede L'Edairage)色彩系统。广泛应用于人造灯光、电影电视、计算等领域。CIE色度图包含了可见光的全部色域，它只是一个二维空间，反应光色的彩度和纯度，没有亮度因素。

# 基于CIE系统的色彩空间

色彩空间（color space）又称色域，是基于CIE色谱的一个子集。

  1. RGB:Adobe RGB色彩空间比sRGB大，色彩显示更好。ProPhoto RGB更大。
  2. CMYK：主要用作印刷。
  3. Lab:Lab基于孟塞尔色立体，是一个三维坐标轴，L表示由黑到白，a为由绿到红，b为由蓝到黄。
  4. YUV：主要用于优化彩色视频信号传输，与RGB相比，传输时只需要极少的频宽。亮度信号为Y（Luminance或Luma），色度信号为UV（Chrominance或Chroma），描述色彩及饱和度。因为人类对低频比高频敏感的多；另一方面，对亮度的改变也比色彩改变敏感的多，所以我们可以抛弃一定的UV分量，控制好Y分量。

# 眼球的构造

过多的色彩刺激容易造成眼部疲劳，进而失去注意力；在影视调色中要注意这一点，适当穿插大远景调节视觉和减小色相差，减少神经疲劳度；亦或是安排一段温馨的剧情和舒缓的背景音乐来舒缓疲劳。

# 细胞对光的编码

人眼感知明暗的能力远胜过于对色彩的感知能力。对于夜间的场景，由于锥状细胞感受不到太多色彩，所以要适当降低饱和度，增加画面明暗层次。

# 大脑对视觉信号的编码

大脑相对来说是调色而不是校色；调色的因素为情感，例如说看见喜欢的女孩穿白衣服会提高白衣服的亮度。根据此，会引申出声色蒙太奇手法。
